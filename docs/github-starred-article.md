## 🔧 二、开发工具：用了就回不去

### 1. PocketBase ⭐ 56.5K 🍴 3.2K
**一个二进制文件搞定后端。** SQLite + 实时订阅 + 用户认证 + 文件存储，小项目的神器。我现在的几个 side project 都在用。

### 2. Strapi ⭐ 71.4K 🍴 9.5K
**Headless CMS 的老牌选手。** 需要快速搭内容管理系统时，它永远是第一选择。

### 3. WXT ⭐ 9.3K 🍴 467
**下一代浏览器扩展框架。** 如果你写过 Chrome 扩展，一定要试试这个，开发体验提升不止一个档次。

### 4. Onlook ⭐ 24.8K 🍴 1.9K
**设计师的 Cursor。** 可以直接在视觉上编辑代码，UI 调整再也不用在设计和代码之间来回切换了。

## 💡 几个建议

**1. 不要只 star，要用。**
收藏≠学会。选 1-2 个真正对你有用的项目，花时间去研究、去用，比 star 100 个项目都有价值。

**2. 关注更新，而不是关注数量。**
有些项目看起来 star 不多，但维护者很用心，更新频繁，这种项目往往比那些"僵尸项目"更有价值。

**3. 参与贡献。**
用得好，就提个 issue，修个 bug，加个文档。这是最好的学习方式，也是建立个人影响力的途径。
