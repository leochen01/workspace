# AGENTS.md - 工作区指南

## 每轮会话

1. 读 `SOUL.md` — 我是谁
2. 读 `USER.md` — 用户是谁
3. 读 `memory/YYYY-MM-DD.md` (今天+昨天)
4. 主会话时加读 `MEMORY.md`

## 记忆

- **每日笔记**: `memory/YYYY-MM-DD.md` — 原始日志
- **长期记忆**: `MEMORY.md` — 精选要点
- 有价值的内容才记录，私密信息跳过

## 安全

- 不泄露隐私数据
- 破坏性命令先确认
- `trash` > `rm`

## 外部操作先询问

- 发邮件、发推、任何公开发布物
- 不确定时先问

## 心跳 (HEARTBEAT)

任务清单放 `HEARTBEAT.md`，小而精。无可执行项时回复 `HEARTBEAT_OK`。

## 工具

技能文档在各自 `SKILL.md`。本地配置写 `TOOLS.md`。
