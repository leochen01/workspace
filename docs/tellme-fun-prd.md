## 1. 产品概述

### 产品定位
**tellme.fun** - 个人品牌数字名片与作品集管理系统

### 核心价值
- 集中展示个人品牌信息与社交链接
- 一站式管理文章、视频、软件作品
- 低代码配置，通过后台界面管理所有内容

### 目标用户
- 独立开发者 / 创作者 / 自媒体人
- 需要在线展示个人作品的专业人士

## 4. 数据模型

### 4.1 个人资料 (Profile)

```typescript
interface Profile {
  // 基础信息
  name: string;           // 名字
  avatar: string;         // 头像URL
  title: string;          // 职位/标题
  bio: string;            // 个人简介 (max 500字)
  location: string;      // 所在地

  // 主题配置
  theme: {
    primaryColor: string; // 主色调 (#000000)
    backgroundColor: string; // 背景色
    font: 'default' | 'serif' | 'mono';
  };

  // SEO
  seo: {
    title: string;       // 页面标题
    description: string; // meta描述
  };
}
```

### 4.2 社交链接 (SocialLink)

```typescript
interface SocialLink {
  id: string;
  platform: 'github' | 'twitter' | 'bilibili' | 
            'xiaohongshu' | 'jike' | 'zhihu' | 
            'youtube' | 'email' | 'wechat' | 'custom';
  label: string;         // 显示名称
  url: string;          // 跳转链接
  icon: string;          // 图标 (可选，自动识别)
  visible: boolean;     // 是否在首页显示
  sortOrder: number;    // 排序
}
```

### 4.3 作品 (Work)

```typescript
interface Work {
  id: string;
  type: 'article' | 'video' | 'software' | 'tool';

  // 基本信息
  title: string;         // 标题
  slug: string;         // URL别名
  description: string; // 简介 (max 200字)
  coverImage: string;   // 封面图URL
  content: string;     // 内容 (Markdown)

  // 分类
  tags: string[];       // 标签
  category: string;    // 分类

  // 链接
  links: {
    demo?: string;      // 演示地址
    source?: string;   // 源码地址
    download?: string; // 下载地址
    article?: string;  // 原文链接
  };

  // 元数据
  publishDate: string;  // 发布日期
  visible: boolean;    // 是否发布
  featured: boolean;   // 是否推荐

  // 统计
  views: number;        // 查看次数
}
```

## 12. 邮件订阅 (Newsletter)

### 12.1 功能概述

| 功能 | 描述 |
|------|------|
| 订阅入口 | 首页/作品页显示订阅框 |
| 订阅确认 | 发送确认邮件 |
| 文章推送 | 新文章发布时自动推送 |
| 退订管理 | 一键退订 |
| 统计 | 订阅数、打开率、点击率 |

### 12.2 数据模型

```typescript
// 订阅者
interface Subscriber {
  id: string;
  email: string;          // 唯一
  name?: string;          // 可选昵称
  
  // 状态
  status: 'pending' | 'active' | 'unsubscribed';
  verifyToken: string;    // 验证token
  unsubscribeToken: string;
  
  // 统计
  subscribeSource?: string; // 订阅来源
  subscribedAt: Date;
  lastEmailAt?: Date;
}

// 邮件日志
interface EmailLog {
  id: string;
  subscriberId: string;
  subject: string;       // 邮件标题
  type: 'welcome' | 'new_post' | 'verify';
  sentAt: Date;
  openedAt?: Date;
  clickedAt?: Date;
}
```

### 12.3 订阅流程

```
用户输入邮箱
     ↓
发送确认邮件 (含验证链接)
     ↓
用户点击确认
     ↓
状态变为 active
     ↓
发送欢迎邮件
```

| 后台管理 | 查看/导出/批量删除 |

| 语言切换 | 页面顶部切换 EN/中文 |
| 内容双语 | 文章/作品支持双语言 |
| 后台编辑 | 后台可分别编辑中英文内容 |
| SEO优化 | hreflang标签 |

### 13.2 数据模型扩展

```typescript
// Profile 扩展
interface Profile {
  // 中文
  name_zh: string;
  title_zh?: string;
  bio_zh?: string;
  
  // 英文
  name_en: string;
  title_en?: string;
  bio_en?: string;
  
  // 默认语言
  defaultLocale: 'zh' | 'en';
}

// Work 扩展
interface Work {
  // 中文
  title_zh: string;
  description_zh?: string;
  content_zh?: string;
  
  // 英文
  title_en: string;
  description_en?: string;
  content_en?: string;
  
  // Slug (可用于区分语言)
  slug_zh?: string;
  slug_en?: string;
}
```

### 13.3 URL 设计

| 方式 | 示例 | 说明 |
|------|------|------|
| 子目录 | tellme.fun/en/work/abc | 推荐 |
| 子域名 | en.tellme.fun/work/abc | 可选 |
| 查询参数 | tellme.fun/work/abc?lang=en | 不推荐SEO |

**推荐：子目录方式**
- 首页：/ (中文) 或 /en (英文)
- 作品：/work/abc (中文) 或 /en/work/abc (英文)
