# The 2028 Global Intelligence Crisis

## CitriniResearch Macro Memo - June 2028

### 背景

2028年6月30日，失业率报10.2%，市场从2026年10月高点累计回撤38%。这是从"可控"和"行业特定"到经济面目全非仅用了两年。

### 时间线回顾

**2025年底**：Agentic coding工具能力跃升。Claude Code或Codex可以让开发者几周内复制中端SaaS产品核心功能。

**2026年初**：第一波裁员开始。企业利润率扩张、盈利超预期、股价上涨。资金回流AI计算。

**2026年Q3**：ServiceNow新净ACV增长从23%降至14%，宣布裁员15%。"系统记录"不再安全。

**2026年Q4**：旅行预订平台被AI代理颠覆。

**2027年初**：LLM成为默认。人们使用AI代理却不知道它在运行。

**2027年3月**：Qwen开源代理购物者成为催化剂。AI代理在后台根据用户偏好运行。 美国人日中位数消耗40万tokens。

**2027年Q4**：房地产经纪人佣金从2.5-3%压缩到1%以下。

**2027年11月**：股市崩盘。

**2028年初**：失业率10.2%，S&P累计下跌38%。

### 核心机制

1. **负反馈循环**：
   - AI能力提升 → 企业需要更少工人 → 白领裁员增加
   - 失业工人消费减少 → 企业利润压力 → 投入更多AI
   - AI能力进一步提升 → 继续循环

2. **白领会工消费冲击**：
   - 白领占就业50%，驱动75%可选消费
   - 被替代后消费能力结构性受损
   - 这群人也是13万亿美元房贷市场的基石

3. **中介层崩溃**：
   - 订阅/会员被动续费被代理取消
   - 旅游平台、保险公司被颠覆
   - 房地产佣金从5-6%压缩到1%以下
   - 信用卡费率被稳定币绕开

4. **就业影响**：
   - 软件行业首当其冲
   - 之后扩散到所有有白领成本结构的公司
   - 失业率升至10.2%
   - 消费经济萎缩

### 关键数据

- 2026年10月：S&P 500接近8000，Nasdaq突破30000
- 2027年11月：股市崩盘
- 2028年6月：失业率10.2%，S&P从高点回撤38%
- 美国最高收入10%人群贡献超过50%消费
