# 总刷到同款推荐？你可能掉进了「注意力偏差」的陷阱

有没有过这种经历：想挑个防晒，刷1小时小红书全是同一款“爆款”，咬咬牙买了，结果闷痘到怀疑人生；职场上，领导总夸天天加班的同事“最努力”，却没看到默默高效完成3个项目的你？其实不是你眼光差，也不是领导偏心——你们都掉进了「注意力偏差」的坑里。

注意力偏差，简单说就是大脑的“偷懒机制”：面对一堆信息时，我们会自动优先关注**熟悉、显眼、符合预期**的内容，直接过滤掉“陌生但重要”的选项。
- 消费场景：你刷到10次同一款防晒，大脑会默认“大家都买=好用”，自动忽略“这款适合油皮但你是干皮”的关键信息；
- 职场场景：领导每天下班都看到A员工加班，大脑会把“加班”和“努力”绑定，完全没注意到B员工提前3天就提交了高质量方案。

这是进化留下的本能——原始人只需要关注熟悉的猎物/危险就能活下去，不用费脑分析所有可能性。但现在环境变复杂了，这种“省力机制”反而成了决策bug：它让我们活在“熟悉的信息茧房”里，错过更优解。

不用逼自己“时刻保持理性”，给大脑装3个小补丁就行：
1. **列「反向清单」**：买东西前，写3个“不选热门款的理由”（比如“爆款香精多”“不适合我的肤质”）；职场评估时，写2个“这个员工可能存在的问题”，强制自己看到另一面；
2. **设「强制观察期」**：每天抽5分钟，刻意关注1个“平时没注意的信息”——比如刷手机时跳过前3条推荐，翻到第10条；开会时，特意问沉默的同事“你有什么想法？”；
3. **换「信息源」**：每周关注1个和你风格完全不同的账号（比如你总看美妆，关注1个极简护肤号）；职场上，换个角度看问题：比如从“谁加班多”换成“谁的成果转化率高”。

注意力偏差不是你的错，但放任它，你可能会错过适合自己的防晒、被低估的能力，甚至更棒的人生选项。下次再被“熟悉的信息”牵着走时，记得给大脑提个醒：“嘿，别偷懒，看看另一边～”
