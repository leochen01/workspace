# QQ说说回忆展示应用 - 设计方案

## 1. 项目概述

**项目名称**: QQ说说回忆馆 (QQMoments Memory)  
**类型**: 本地Web应用（单页应用）  
**核心功能**: 展示QQ说说内容，以震撼的视觉效果呈现回忆主题  
**目标用户**: 希望保存和展示QQ说说回忆的用户

## 2. 数据格式

### 2.1 导入数据格式 (JSON)

```json
{
  "user": {
    "qq": "123456789",
    "nickname": "用户的QQ昵称"
  },
  "moments": [
    {
      "id": "unique_id",
      "content": "说说内容文字",
      "images": ["image1.jpg", "image2.jpg"],
      "created_at": "2015-07-15 18:30:00",
      "like_count": 15,
      "comment_count": 3,
      "comments": [
        {"content": "评论内容", "author": "评论者"}
      ],
      "location": "北京市",
      "mood": "happy" 
    }
  ]
}
```

### 2.2 数据获取方式

用户提供数据的方式（应用本身不提供爬取功能）：
- 使用现有QQ说说导出工具导出JSON
- 手动整理CSV/JSON文件
- 提供图片文件夹

## 3. 视觉与交互设计

### 3.1 整体风格

- **暗色主题**: 深色背景配柔和光效，营造沉浸式回忆感
- **时光隧道**: 以时间线为主轴，带用户穿梭回忆
- **震撼动画**: 流畅的过渡效果，图片幻灯片，视频背景

### 3.2 页面结构

#### 启动页
- 全屏背景：用户第一张说说配图或抽象光效动画
- 大标题："回忆的力量" / "十年时光"
- 副标题：QQ号 + "从2012到2024"
- 进入按钮：带有光晕效果的"进入回忆"

#### 时间隧道（首页）
- 3D纵向时间线，从右向左流动
- 每个节点是一个年份/月份
- 节点悬停：显示该时间段的亮点（说说数量、最佳照片）
- 滚动时有粒子飘落效果

#### 说说详情页
- 大图轮播（支持手势滑动）
- 文字内容配以优雅的字体动画
- 点赞数、评论数可视化展示
- 评论区以对话气泡形式呈现

#### 年份总览页
- 该年说说照片拼贴墙（瀑布流/马赛克）
- 年度关键词词云
- 说说数量、照片数量统计卡片
- 热门说说Top 10

#### 回忆地图（可选）
- 如果有位置信息，在地图上标记
- 使用暗色地图样式

### 3.3 动画效果

- **页面切换**: 淡入淡出 + 轻微位移
- **图片加载**: 渐进式加载 + 模糊到清晰
- **时间线滚动**: 视差滚动效果
- **文字出现**: 打字机效果或逐行淡入
- **特殊日期**: 春节、生日等有烟花/彩带特效
- **背景**: 缓慢移动的星空/光点/粒子

### 3.4 音效（可选）

- 背景音乐：温暖的钢琴曲/回忆向音乐
- 点击音效：轻微的"滴"声
- 翻页音效：纸张翻动声

## 4. 技术实现

### 4.1 技术栈

- **框架**: 纯HTML + CSS + JavaScript（无需构建工具）
- **动画库**: 原生CSS动画 + 少量JS
- **图片**: 原生img + object-fit优化
- **本地运行**: 双击index.html即可

### 4.2 目录结构

```
qq-memory/
├── index.html          # 主页面
├── css/
│   └── style.css       # 样式
├── js/
│   └── app.js          # 逻辑
├── data/
│   └── moments.json    # 说说数据
└── images/             # 说说图片
    └── (用户图片)
```

### 4.3 核心功能模块

1. **数据解析器**: 读取并解析JSON数据
2. **时间线引擎**: 生成时间线DOM
3. **图片懒加载**: 优化大量图片性能
4. **动画控制器**: 管理页面切换动画
5. **本地存储**: 记住用户浏览位置

## 5. 快速开始

1. 创建一个文件夹 `qq-memory`
2. 将 `index.html`, `css/style.css`, `js/app.js` 放入
3. 创建 `data/moments.json` 导入数据
4. 将图片放入 `images/` 文件夹
5. 双击 `index.html` 打开
