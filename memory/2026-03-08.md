### 常见视频风格特征

#### 1. 画面布局
- 📐 **分屏设计**: 左侧/上方显示视频素材，右侧/下方显示文字讲解
- 🎨 **动态字幕**: 关键词高亮显示，配合动画效果
- 🖼️ **画中画**: 讲师头像小窗口 + 主要内容区域
- 📊 **信息图表**: 简洁的图标、流程图、对比表

#### 2. 视觉元素
- 明亮清新的配色（蓝/绿/橙为主）
- 大号易读字体（Arial/Helvetica/思源黑体）
- 平滑的转场动画（淡入淡出、滑动）
- Emoji 和图标点缀

#### 3. 音频特点
- 清晰的旁白配音
- 轻柔的背景音乐
- 音效点缀（提示音、转场音）

### 📝 制作流程

#### 1️⃣ 内容策划
```
选题 → 脚本撰写 → 分镜设计
```

#### 2️⃣ 素材准备
- 📹 录制视频/音频
- 🖼️ 收集图片、图标
- 📄 准备文字内容

#### 3️⃣ 剪辑制作（以剪映为例）
1. 导入主视频轨道
2. 添加背景图层
3. 插入文字/字幕（动画效果）
4. 添加转场（0.3-0.5 秒淡入淡出）
5. 配乐 + 音效
6. 调色（明亮、清新风格）
7. 导出 1080p/4K

#### 4️⃣ 封面制作
- 使用 Canva 模板
- 大标题 + 醒目配色
- 统一频道视觉风格
